# Getting Started

`phenotype-go-kit` is a Go infrastructure toolkit extracted from the Phenotype ecosystem. It provides small, focused packages for logging, data structures, polling, and registries.

## Installation

```bash
go get github.com/KooshaPari/phenotype-go-kit
```

## Overview

See the [README](https://github.com/KooshaPari/phenotype-go-kit) for the full package list and documentation.
